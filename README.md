# 🎨 PixelPickr – Smart Color Picker Chrome Extension

> Extract colors from any website instantly 🚀

---

## 🧠 Problem Statement

Developers and designers often struggle to identify exact colors used on websites.
Manually inspecting elements or using external tools breaks workflow and slows productivity.

---

## 🚀 Solution

**PixelPickr** is a lightweight Chrome Extension that allows users to:

* 🎯 Pick colors directly from any webpage
* 📋 Copy HEX / RGB values instantly
* ⚡ Get real-time color preview
* 🧩 Improve UI/UX workflow without leaving the browser

---

## ✨ Features

* 🎨 One-click color picker
* 🖱️ Hover to detect colors dynamically
* 📋 Copy to clipboard support
* 💡 Clean and minimal UI
* ⚡ Fast and lightweight (no external dependencies)

---

## 🏗️ Architecture

```
chrome-extension/
│
├── manifest.json        # Extension configuration
├── popup.html           # UI layout
├── popup.css            # Styling
├── popup.js             # Logic handling
├── content.js           # Interacts with webpage
└── assets/              # Icons & images
```

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Chrome Extension APIs

---

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/PixelPickr.git
```

2. Open Chrome and go to:

```
chrome://extensions/
```

3. Enable **Developer Mode**

4. Click **Load Unpacked**

5. Select the project folder
